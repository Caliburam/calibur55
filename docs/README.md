## What is Calibur?

Calibur is an AI-powered gamified self-improvement platform.

Instead of tracking habits like a traditional todo app,
Calibur turns your personal growth into an RPG.

Users complete real-life missions, earn XP, level up,
unlock achievements, build streaks, and gradually create
their own stronger version.

Every layer has only one responsibility.
